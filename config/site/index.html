dbpatcher

dbpatcher is a tool for applying incremental updates to a MySQL database.

It has originally developed for the continuous deployment procedure of Nextdoor, a site developed by Huitale Oy. It was used for the automated nightly deployment to production procedure.

In agreement with Huitale Oy the code has been forked and made open source. It is currently maintained and developed by its original author.
